Django[
    创建项目:	django-admin startproject （name）
    创建App:  	python manage.py startapp （name）
    创建超级管理员：	    python manage.py createsuperuser
    生成数据库迁移文件：	python manage.py makemigrations
    执行数据库执行文件：	python manage.py migrate
]
生成项目依赖文件：pip freeze > requirements.txt
安装项目依赖文件：pip install -r requirement.txt
查看端口：netstat -an
